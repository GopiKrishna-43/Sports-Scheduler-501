# Sports-Scheduler-501

## Description
The Sports Scheduler is an application designed to automate the scheduling and management of sports events. It helps organize matches, assign teams or players, and manage match timings efficiently while reducing manual work and scheduling conflicts.

## Objectives
- To simplify sports event scheduling
- To reduce conflicts in match timings
- To provide an easy-to-use system for managing sports events
- To improve accuracy and efficiency in scheduling

## Features
- Create and manage sports events
- Schedule matches with date and time
- Team or player management
- View upcoming and completed matches
- User-friendly interface

## Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js with Express  
- Database: MongoDB / MySQL (optional)
